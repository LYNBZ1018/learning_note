# Java 复习

* 方法可以分为
  * 实例方法
  * 静态方法 静态方法 只能调用静态方法使用静态变量

* static 加{} 静态语句
  * 语句必须在一个方法中或静态语句中
* 可变参数
  * 放在最后、若干参数类型相同,可变参数用“...”表示
  * public void f(int x, int y, int ...x);
* this
  * 出现在方法中，表示调用这个方法的对象
